Mac装机指南

一、登陆、激活

  • 是否需要登陆Apple ID
  • 用户名：fc

二、基础设置

  • 系统设置

    • Apple ID
      • 如果需要iCloud空间，则转回国区
    • 主要设置
      • 程序坞
        • 自动隐藏、右置、大小、指示灯
      • 调度中心
        • 窗口分组、显示桌面、触发角（触发屏保不要带快捷键）
      • 网络
        • DNS
      • 键盘
        • 清空快捷键、大小写切换输入法
      • 共享
        • 修改电脑名字

  • 命令设置

    • 启动trim

      sudo nvram boot-args=rootless=0 #（作用：关闭苹果最新引入的Rootless安全机制）
sudo trimforce enable
#忽视警告 `Y`，重新启动 `Y`
sudo nvram -d boot-args #（作用：重新打开苹果的Rootless安全机制）

    • 长按连续输入

      defaults write NSGlobalDomain ApplePressAndHoldEnabled -boolean false

    • finder显示完整路径

      defaults write com.apple.finder _FXShowPosixPathInTitle -bool true;killall finder #显示
defaults write com.apple.finder _FXShowPosixPathInTitle -bool false ;killall finder #隐藏

    • finder始终显示用户资料库

      chflags nohidden /Library/ #显示
chflags hidden /Library/ #隐藏

    • 取消chrome 左右滑 前进后退

      defaults write com.google.Chrome AppleEnableSwipeNavigateWithScrolls -bool false

    • 使用全部来源软件

      sudo spctl --master-disable

    • 终端隐藏命令：

      # 显示
defaults write com.apple.Dock showhidden -bool TRUE;
killall Dock
# 隐藏
defaults write com.apple.Dock showhidden -bool FALSE;
killall Dock

    • 开启一位数密码终端代码

      pwpolicy -clearaccountpolicies

    • App Store去除小红点（未成功）
      进入~/Library/Preferences/
      用Xcode（或者其他任何Plist编辑器）打开 com.apple.appstored.plist，修改其中appStoreBadgeCount的值为0。
      注销或者重启即可。

    • 系统偏好 去除小红点（重启会重新显示）

      defaults write com.apple.systempreferences AttentionPrefBundleIDs 0
killall Dock

    • 设置语言为英文

      locale #查看当前使用语言
sudo languagesetup #回车输入密码后显示语言列表，输入对应序号后回车即可设置

      （ps：mac分为本地域语言（主语言）和用户域语言（次语言），一般情况下你登录了账户，在偏好设置里面改的语言是“次语言”，不包括登录界面等系统级的显示语言，而使用以上终端修改的是整个系统的所有语言显示，最后，记得要重启。）

四、快捷键收集

  • 常用快捷键
    • Command-V：将剪贴板的内容粘贴到当前文稿或应用中。这同样适用于“访达”中的文件。
    • Option-Command-V：将剪贴板中的文件从原始位置移动到当前位置。
    • Control-Command-Q：立即锁定屏幕。
    • Shift–Command–Q：退出登录您的 macOS 用户帐户。系统将提示您确认。
    • Option-Shift-Command-Q：退出登录您的 macOS 用户帐户且不需要确认。
  • 系统快捷键
    • Fn-上箭头：Page Up：向上滚动一页。
    • Fn–下箭头：Page Down：向下滚动一页。
    • Fn–左箭头：Home：滚动到文稿开头。
    • Fn–右箭头：End：滚动到文稿末尾。
    • Shift–Command–句号：finder显示隐藏文件夹
    • Option-Command-C：finder复制路径
    • Option-Command-Esc：强制退出应用。
    • Control-Command–空格键：显示字符检视器，您可以从中选择表情符号和其他符号。
    • 空格键：使用“快速查看”来预览所选项。
    • Option-空格键：显示所选文件的快速查看幻灯片显示。
  • 文稿快捷键
    • Control–A：移至行或段落的开头。
    • Control–E：移至行或段落的末尾。
    • Control–F：向后移动一个字符。
    • Control–B：向前移动一个字符。
    • Control–P：上移一行。
    • Control–N：下移一行。
    • Control–L：将光标或所选内容置于可见区域中央。
    • Control-K：删除插入点与行或段落末尾处之间的文本。
    • Control-H：删除插入点左边的字符。也可以使用 Delete 键。
    • Control-D：删除插入点右边的字符。也可以使用 Fn-Delete。

五、命令别名添加

在配置文件中添加别名。
zsh配置文件地址

  • 系统级/etc/zshrc
  • 用户级的~/.zshrc
# 编辑配置文件
vim ~/.zshrc
# 添加下面内容
alias ll='ls -alF'
alias la='ls -A'
alias l='ls -CF'
# 刷新配置
source ~/.zshrc
