使用hexo和github搭建个人博客

准备工作

注册GitHub

  • 设置SSH

安装GIT（用于发布到github）

git --version

安装NodeJS（用于安装hexo）

node -v
npm -v

全局安装hexo

  • 安装hexo
npm install -g hexo-cli
hexo version

开始

启动hexo

cd <博客目录>

REM必须是空目录
hexo init
REM 安装package.json中全部依赖
npm install
REM 生成静态页面
hexo g
REM 启动hexo服务[localhost:4000]
hexo s

tips:

hexo init执行了两步操作

  • clone https://github.com/hexojs/hexo-starter
  • 执行 npm install

配置Hexo到Github

cd <博客目录>

修改最下面的配置，可选http和SSH，http需要每次输入密码

deploy:
type: git repository: http://github.com/implcated/implcated.github.io.git
branch: main

迁移

使用不同GitHub分支会使源码仓库为public，本方法需要将之前的源代码建立一个远程仓库。

### 新建hexo项目

在新电脑上创建hexo项目，并执行hexo init、npm install命令

### 移除安装的hexo文件

将hexo项目中除.git node_modules之外的文件全部删除

### 关联远程仓库

点击查看之前的文章

  • clone仓库

  • 执行npm install
